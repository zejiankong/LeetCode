自己的思路：
创建一个新的链表，用两个指针遍历两个旧链表，直到全部遍历完，有点浪费空间

别人的思路：
采用递归的思想，空间复杂度降低。
class Solution {
    public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if (l1 == null) {
            return l2;
        }
        else if (l2 == null) {
            return l1;
        }
        else if (l1.val < l2.val) {
            l1.next = mergeTwoLists(l1.next, l2);
            return l1;
        }
        else {
            l2.next = mergeTwoLists(l1, l2.next);
            return l2;
        }

    }
}
